# Whist

Whist is a small Go web app for tracking scores in [call-ace
Whist](https://en.wikipedia.org/wiki/Call-ace_Whist) (Jyde whist). Players form
clubs, log games and tricks, and the app keeps a leaderboard.

## Installation

Requires [Go](https://go.dev/) 1.25 or newer.

```bash
go install github.com/martinohansen/whist@latest
```

## Usage

Start the server

```bash
whist
```
